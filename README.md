# Тестовое задание для ЭтКонс

## Использованный стек:

1. Vue 3
2. Prime Vue
3. Axios

### Инструкция по установке

1. Скопировать проект из репозитория командой git clone.
2. Установить необходимые зависимости командой npm i.
3. Запустить проект командой vue serve.
